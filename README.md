# orbit-sites-scaffold

Minimal Next.js scaffold for building customer-facing sites via AI agents in Vercel Sandboxes.

## What's included

- **Next.js 16** — App Router with TypeScript (strict)
- **Tailwind CSS 4** — Configured with CSS variables and dark mode support
- **shadcn/ui** — Initialized with `cn()` utility and Button component. Add more with `npx shadcn@latest add [component]`
- **ESLint** — Next.js rules configured
- **Geist font** — Pre-configured via `next/font`

## What's NOT included

No pages, no components, no API routes — just the configured tooling. This is a blank canvas.

## Usage

```bash
npm install
npm run dev
```

See `CLAUDE.md` for agent-facing documentation on project structure and conventions.
